# Analisis-Sistem-Keamanan-Pada-Software-Defined-Network-Dengan-Hybrid-Honeypot
Repository ini berisi source code, dokumentasi, dan file pendukung Tugas Akhir yang dikembangkan sebagai salah satu syarat memperoleh gelar Sarjana Informatika. Proyek ini diharapkan dapat menjadi referensi dalam pengembangan sistem serupa di masa mendatang.

Yang akan di share disini adalah beberapa rules snort, script python untuk push flow rules di SDN dan script untuk mengirimkan log yang dideteksi ke SDN
